About the Data

      

        

          This lab uses annual broad money as a percentage of GDP for the
          Euro area (EMU), sourced from the World Bank Open Data
          indicator FM.LBL.BMNY.GD.ZS. It measures how large the monetary aggregate
          is relative to the size of the economy. [web:35][web:36][web:31]
        

        

          When the World Bank API is not reachable from your browser or this indicator is
          unavailable, the lab switches to a small built‑in demo series so all scenarios still
          run and visualize correctly. The banner above the chart tells you whether you are
          seeing LIVE World Bank data or DEMO fallback.
        

      

    


    

      
How This Lab Works

      

        

          A lightweight DuckDB-WASM database runs inside your browser, and the
          World Bank time series is loaded into it as a table. SQL window functions like
          LAG, AVG, and STDDEV_SAMP compute growth,
          volatility, and moving averages on the fly. [web:80][web:25]
        

        

          Each scenario executes a different SQL query against the same dataset, then updates the
          chart, insight text, and result table so you can see how analytical logic turns raw
          macro data into an investment narrative.
