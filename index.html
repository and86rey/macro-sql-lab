Data Source & Coverage

This lab uses annual GDP per capita (constant 2015 US$) from the World Bank for Germany (DEU). The indicator code is NY.GDP.PCAP.KD. It measures economic output per person, adjusted for inflation, and is widely used to track living standards over time. [web:72]

If the live World Bank API is not reachable from your browser, the lab automatically switches to a small built‑in demo series with similar behavior so that all scenarios still run and visualize correctly.

How This Lab Works

A lightweight DuckDB-WASM database runs entirely inside your browser (no server, no backend). The lab loads macroeconomic time series into DuckDB and then executes analytical SQL scenarios to compute growth, volatility, and moving averages. [web:80][web:25]

Each scenario uses a different window function (such as LAG, AVG, or STDDEV_SAMP) and immediately updates the chart, insight text, and result table so you can see how changes in SQL logic alter the investment story.
