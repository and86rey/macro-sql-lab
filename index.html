About the Data

      

        

          Use the selectors above to choose a country and a
          World Bank indicator. The app then pulls annual time series through
          the official World Bank Indicators API and loads them into a local
          DuckDB-WASM database in your browser.
        

        

          Example indicators include GDP per capita (constant 2015 US$)
          (NY.GDP.PCAP.KD) and real GDP growth (annual %)
          (NY.GDP.MKTP.KD.ZG). These are widely used macro series in investment
          and policy analysis.
        

      

    


    

      
How This Lab Works

      

        

          A lightweight DuckDB-WASM database runs inside your browser.
          Once the World Bank series is loaded as a table, SQL window functions like
          LAG, AVG, and STDDEV_SAMP compute growth,
          volatility, and moving averages in real time.
        

        

          Each scenario executes a different SQL query against the same dataset, then updates the
          chart, insight text, and result table so you can see how analytical logic turns raw
          macro data into an investment narrative.
